## Why

A freelance photographer needs a clean, performant landing page to showcase their work and establish an online presence. Currently there is no web presence — this project creates a simple but elegant single-page portfolio from scratch.

## What Changes

- Scaffold a Vite + TypeScript project in the `Portfolio_Web` directory
- Build a responsive **Navbar** with logo, navigation links, and mobile hamburger menu
- Build a **Showcase/Hero** section with a full-viewport hero image, headline, tagline, and CTA
- Build a **Footer** with social links, copyright, and back-to-top
- Clean, minimal design optimized for photography (dark aesthetic, full-bleed images)
- All components built with vanilla CSS modules (no UI library dependency)

## Capabilities

### New Capabilities
- `portfolio-layout`: Responsive page shell with Navbar, main content area, and Footer
- `hero-showcase`: Full-viewport hero section with background image, overlay text, and call-to-action
- `site-navigation`: Fixed-position Navbar with logo, nav links, and mobile responsive menu
