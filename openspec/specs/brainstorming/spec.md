## Requirements
### Requirement: Brainstorming Skill Availability
The system SHALL make a `brainstorming` skill available in the `.agent/skills/` directory.

#### Scenario: Agent provides brainstorming assistance
- **WHEN** the user invokes or requires the brainstorming skill
- **THEN** the agent reads the skill instructions and provides structured ideation, evaluating multiple alternatives before committing to a solution.
