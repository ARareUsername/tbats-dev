## Why

The TBATS model is a powerful forecasting algorithm, but it is complex and lacks an interactive, visually engaging representation. Creating a high-fidelity landing page with a live interactive simulator will help developers and data scientists understand TBATS' components (Trigonometric seasonality, Box-Cox transformation, ARMA errors, Trend, Seasonal) and how they affect time-series data.

## What Changes

- Create a modern, high-fidelity single-page landing page website for TBATS.
- Implement an interactive SVG time-series decomposition simulator allowing real-time parameter tweaking.
- Provide R and Python code snippets for quick integration.
- Host the application using a React + Vite setup on localhost.

## Capabilities

### New Capabilities
- `landing-page`: A web-based landing page for TBATS forecasting model showcasing its features and providing an interactive simulator.

### Modified Capabilities

## Impact

- Adds a new frontend subproject/application for the landing page.
- Utilizes Vite, React, and standard CSS.
