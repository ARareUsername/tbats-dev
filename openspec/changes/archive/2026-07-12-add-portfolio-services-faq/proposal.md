## Why

The current tbats.dev website is missing critical elements of a premium agency page: showcasing past work (portfolio), detailing specific services and technologies used, and addressing common client inquiries (FAQ). Adding these sections will build trust, educate visitors, and improve conversion rates.

## What Changes

- Create a `Portfolio` component showing high-fidelity project cards with hovering effects.
- Create a `Services` component explaining specialties and technical expertise.
- Create an `FAQ` component utilizing an accordion design with smooth transitions.
- Integrate all three new components into the main landing page layout.

## Capabilities

### New Capabilities
- `portfolio`: Showcase of premium agency projects.
- `services-list`: Outline of services and technology stack.
- `faq-accordion`: Interactive accordion listing common client questions and answers.

### Modified Capabilities
None

## Impact

- Modifies the main application layout (`App.jsx`) to include the new sections.
- Introduces new React components in `src/components/`.
- No breaking changes.
