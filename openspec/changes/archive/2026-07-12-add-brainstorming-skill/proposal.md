## Why

Adding a brainstorming skill enables the agent to assist the user with structured, creative ideation directly within the workflow. This solves the problem of needing dedicated prompts or external tools for exploring ideas before committing to decisions or writing code.

## What Changes

- Introduce a new `brainstorming` skill for the agent.
- Fetch or define the skill configuration (e.g., `SKILL.md`) to integrate it with the agent's capabilities.

## Capabilities

### New Capabilities
- `brainstorming`: A skill that provides structured methodologies for brainstorming, ideation, and exploring alternatives.

### Modified Capabilities
None

## Impact

- Adds a new skill directory (e.g., `.agent/skills/brainstorming/`) with necessary configurations.
- Enhances the agent's problem-solving capabilities without impacting existing features.
